# My Profile - Vue.js

A modern web application developed with **Vue 3**, **TypeScript**, and **Vite**. This project was created to explore Vue's component-based architecture while applying modern frontend development practices.

## Features

- Built with Vue 3 Composition API
- TypeScript support
- Fast development environment powered by Vite
- Modular and maintainable project structure
- Responsive-ready architecture

## Tech Stack

- Vue 3
- TypeScript
- Vite
- HTML5
- CSS3

## Project Structure

```
my-profile-vuejs/

│   ├── App.vue

│   └── style.css
```

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd my-profile-vuejs
```

Install the dependencies:

```bash
npm install
```

## Running the Project

Start the development server:

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## Building for Production

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
